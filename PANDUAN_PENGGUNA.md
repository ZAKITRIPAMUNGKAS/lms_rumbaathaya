## 👨‍💼 Panduan untuk Admin

### **1. Login ke Dashboard Admin**
1. Buka `https://rumbaathaya.id/login`
2. Masukkan email: `admin@rumbaathaya.com`
3. Masukkan password: `password`
4. Klik **Login**

### **2. Menu Dashboard Admin**

#### **📊 Dashboard**
- Statistik jumlah siswa, tutor, materi
- Aktivitas terbaru
- Grafik pertumbuhan siswa

#### **👥 Manajemen Siswa**
- **Lihat Daftar Siswa:** Menu "Siswa"
- **Tambah Siswa Baru:**
  1. Klik tombol "+ Tambah Siswa"
  2. Isi form: Nama, Kelas, Program, Kontak Orang Tua
  3. Upload foto (opsional)
  4. Klik "Simpan"
- **Edit/Hapus Siswa:** Klik ikon pensil/sampah di tabel

#### **👨‍🏫 Manajemen Tutor**
- **Lihat Daftar Tutor:** Menu "Tutor"
- **Tambah Tutor Baru:**
  1. Klik "+ Tambah Tutor"
  2. Isi: Nama, Email, Password, Mata Pelajaran
  3. Upload foto profil
  4. Klik "Simpan"

#### **📅 Manajemen Jadwal**
- **Buat Jadwal Belajar:**
  1. Menu "Jadwal" → "+ Tambah Jadwal"
  2. Pilih Tutor, Siswa, Mata Pelajaran
  3. Tentukan Hari & Jam
  4. Klik "Simpan"

#### **📚 Manajemen Materi**
- **Upload Materi Belajar:**
  1. Menu "Materi" → "+ Upload Materi"
  2. Isi judul, deskripsi, mata pelajaran
  3. Upload file PDF/DOC atau link video
  4. Klik "Simpan"

#### **📝 Blog "Sahabat RA"**
- **Buat Artikel Baru:**
  1. Menu "Posts" → "+ Buat Artikel"
  2. Isi judul, kategori (Kabar Rumba/Karya Siswa/Info)
  3. Tulis konten dengan editor Summernote
  4. Upload gambar thumbnail
  5. Pilih status: Draft/Published
  6. Klik "Publish"

#### **📸 Dokumentasi**
- **Upload Foto/Video Kegiatan:**
  1. Menu "Dokumentasi" → "+ Tambah"
  2. Pilih tipe: Foto atau Video
  3. Upload file atau paste link YouTube
  4. Isi judul, deskripsi, kategori
  5. Tentukan tanggal kegiatan
  6. Klik "Simpan"

#### **⭐ Testimoni**
- **Tambah Testimoni:**
  1. Menu "Testimoni" → "+ Tambah"
  2. Isi nama orang tua, role (Orang Tua Siswa Kelas X)
  3. Tulis testimoni
  4. Rating (1-5 bintang)
  5. Upload foto (opsional)
  6. Klik "Simpan"

#### **📋 Pendaftaran**
- **Kelola Pendaftaran Baru:**
  1. Menu "Pendaftaran"
  2. Lihat daftar calon siswa yang mendaftar online
  3. Ubah status: Pending → Approved/Rejected
  4. Tambahkan catatan jika perlu

## 🔧 Tips & Troubleshooting

### **Lupa Password**
1. Klik "Lupa Password?" di halaman login
2. Masukkan email terdaftar
3. Cek email untuk link reset password
4. Buat password baru

### **Upload Gambar Tidak Muncul**
- Pastikan ukuran file < 2MB
- Format yang didukung: JPG, PNG, WEBP
- Clear cache browser (Ctrl+Shift+R)

### **Error 500 / Halaman Tidak Muncul**
- Clear cache aplikasi:
  ```bash
  php artisan cache:clear
  php artisan config:clear
  php artisan view:clear
  ```
- Hubungi admin sistem

### **Tidak Bisa Login**
- Pastikan email & password benar
- Cek CAPS LOCK
- Gunakan akun default jika lupa:
  - Admin: `admin@rumbaathaya.com` / `password`
  - Tutor: `tutor@rumbaathaya.com` / `password`

## 🔒 Keamanan

### **Best Practices:**
1. ✅ Ubah password default segera setelah login pertama
2. ✅ Gunakan password yang kuat (min. 8 karakter, kombinasi huruf & angka)
3. ✅ Jangan share password ke orang lain
4. ✅ Logout setelah selesai menggunakan sistem
5. ✅ Jangan login di komputer publik/warnet

### **Password yang Kuat:**
- Minimal 8 karakter
- Kombinasi huruf besar & kecil
- Angka
- Karakter khusus (@, #, $, dll)
- Contoh: `Rumba@2024!`

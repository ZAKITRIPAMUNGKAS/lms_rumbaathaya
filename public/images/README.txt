INSTRUKSI: TAMBAHKAN GAMBAR hero-student.png

Untuk memperbaiki masalah loading dashboard yang lambat, silakan:

1. Download gambar 3D siswa dari link berikut (atau gunakan gambar PNG transparan lainnya):
   https://cdn3d.iconscout.com/3d/premium/thumb/boy-reading-book-4897486-4077963.png

2. Simpan gambar tersebut di folder ini (public/images/) dengan nama:
   hero-student.png

3. Jika link di atas tidak bisa dibuka, gunakan gambar PNG transparan apa saja yang ada di komputer Anda.

4. Setelah gambar ditambahkan, refresh halaman dashboard dan masalah loading akan teratasi.

CATATAN:
- Format file: PNG (disarankan dengan background transparan)
- Ukuran: Tidak ada batasan, tapi disarankan tidak terlalu besar (max 1MB)
- Nama file HARUS: hero-student.png (huruf kecil semua)

Setelah menambahkan gambar, hapus file README.txt ini jika sudah tidak diperlukan.
